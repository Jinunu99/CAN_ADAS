## 개요
감속 상황을 후방운전자가 인지할 수 있도록 개선한 브레이크등 제어기,  
원페달 드라이빙 중 가속 페달을 완전히 떼지 않아도 감속에 따라 브레이크등이 점등될 수 있도록 설계  

  
* 개발 기간 : 2024.03.04 ~ 06.07
* 프로젝트 인원 : 1명
